# QR Code Simple - 发展规划

## 核心功能扩展

### 高优先级
- [ ] **全条码支持** - 除二维码外，支持 EAN/UPC/Code 128/Code 39 等一维码识别与生成
- [ ] **多码制识别** - Data Matrix, Aztec, PDF417, MaxiCode
- [ ] **智能内容解析** - 自动识别 WiFi/联系人/日历/邮件/URL/地理位置等格式，提供一键操作（如直接连接 WiFi）
- [ ] **批量生成** - 从 CSV/Excel 导入数据批量生成二维码
- [ ] **样式定制** - 前景色/背景色自定义、中心 Logo、圆角/点阵样式

### 中优先级
- [ ] **二维码修复** - 对模糊/破损二维码的智能修复识别
- [ ] **离线增强模型** - 本地 ML 模型提升低质量图片识别率
- [ ] **扫描区域限定** - 框选特定区域识别，支持多码同时识别高亮

---

## 历史记录增强

- [ ] **全文搜索** - 按内容/时间/类型搜索历史
- [ ] **智能分类** - 自动归类为链接、文本、WiFi、联系人等
- [ ] **收藏/置顶** - 重要内容标记
- [ ] **标签系统** - 自定义标签管理
- [ ] **导入/导出** - JSON/CSV 格式备份，支持云端同步（可选）

---

## 扫描体验优化

- [ ] **连续扫描模式** - 无需确认连续扫描多个码
- [ ] **扫描反馈** - 震动效果
- [ ] **智能对焦** - 根据码大小自动调整对焦距离
- [ ] **手动对焦** - 点击画面中的点进行对焦操作

---

## 分享与导出

- [ ] **矢量导出** - SVG 格式二维码导出（无损放大）
- [ ] **分享模板** - 生成带说明文字的分享图片

---

## 安全与隐私

- [ ] **恶意链接检测** - 集成 VirusTotal 或本地黑名单检测 URL
- [ ] **隐私模式** - 无痕扫描（不保存历史）
- [ ] **应用锁** - 指纹/密码保护敏感历史
- [ ] **本地加密** - 历史数据库加密存储

---

## 界面与体验

- [ ] **Material You 动态主题** - Android 12+ 动态取色
- [ ] **横屏适配** - 平板和横屏优化布局
- [ ] **快捷方式** - 长按图标快速扫码/生成
- [ ] **小组件** - 桌面快捷生成/扫描小组件
- [ ] **动画优化** - 页面过渡、扫描线动画

---

## 技术债务

- [ ] **测试覆盖** - Unit Test + UI Test
- [ ] **性能优化** - 大图处理内存优化、扫描帧率优化
- [ ] **崩溃监控** - 集成 Firebase Crashlytics
- [ ] **国际化完善** - 支持更多语言（日语、韩语、德语等）

---

## 建议开发顺序

1. **智能内容解析**（WiFi/联系人一键操作）- 提升核心体验
2. **全条码支持** - 扩展使用场景
3. **样式定制 + 批量生成** - 满足个性化需求
4. **历史搜索 + 分类** - 数据管理优化
5. **Material You + 横屏适配** - 体验完善
